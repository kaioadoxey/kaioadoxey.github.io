Nested Lists

Lists are tools that allow you to organize your content.

Nested lists allow you to further organize your content, not just in list forms, but also subcategorized lists.

Nested lists are a lot like regular unordered lists because the use the same opening and closing tags.

The difference in a nested list is beginning an unordered list inside of a list item tag.


"ul" This signifies the beginning of an unordered list.


"li" This is the tag allows for a list item.

Description List

Description lists allow you to organize your content into a list form.

It also allows you to explain about each item in the list.

The tags associated with description lists are as follows

"dl" This is the opening and ending description list tag.

"dt" This is the description topic opening and ending tag.

"dd" This is the opening and closing tag for the description of a description topic.
